## Quiz Learning Platform

A web-based quiz and learning platform built with **React, TypeScript, and Firebase** that allows users to practice questions, track their performance, and review incorrect answers.

### Features
- Secure login with **Firebase Authentication**
- Randomized quizzes with shuffled answer options
- Performance tracking and quiz history
- Weak question review with explanations
- Admin panel to manage topics, quizzes, and questions

### Tech Stack
- **Frontend:** React + TypeScript
- **Backend:** Firebase (Firestore + Authentication)
- **Hosting:** Firebase Hosting
